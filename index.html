🔁 For циклі дегеніміз не?

For циклі — белгілі бір тізбек (список, жол, range) бойынша қайталанатын цикл. Алдын-ала неше рет орындалатыны белгілі болғанда қолданылады.

for айнымалы in тізбек: орындалатын_код
💡 Мысал: for i in range(5) дегені — i 0-ден 4-ке дейін ретімен өтеді.

Мысалдар

# Сандар бойынша for i in range(1, 6): print(i, end=' ') # Нәтиже: 1 2 3 4 5 # Тізім бойынша жемістер = ['алма', 'банан', 'алмұрт'] for жеміс in жемістер: print(жеміс) # Жол бойынша for әріп in 'Python': print(әріп, end='-') # Нәтиже: P-y-t-h-o-n-
СинтаксисМағынасы
range(5)0, 1, 2, 3, 4
range(1, 6)1, 2, 3, 4, 5
range(0, 10, 2)0, 2, 4, 6, 8
range(5, 0, -1)5, 4, 3, 2, 1

🔄 While циклі

While циклі — шарт орындалып тұрған уақытта қайталана береді. Неше рет орындалатыны алдын-ала белгісіз болғанда қолданылады.

while шарт: орындалатын_код
⚠️ Назар аудар: Шарт ешқашан жалған болмаса — шексіз цикл болады! Цикл ішінде шартты өзгертуді ұмытпа.

Мысалдар

# Негізгі while циклі санауыш = 1 while санауыш <= 5: print(санауыш) санауыш += 1 # Нәтиже: 1 2 3 4 5 # Пайдаланушы енгізгенше жауап = '' while жауап != 'шығу': жауап = input('Команда енгіз: ') print(f'Сен енгіздің: {жауап}')

⛔ Break операторы

Break — циклді дереу тоқтатады. Шарт орындалған сәтте цикл аяқталады.

for i in range(10): if i == 5: break # 5-ке жеткенде тоқта print(i, end=' ') # Нәтиже: 0 1 2 3 4

⏭️ Continue операторы

Continue — ағымдағы итерацияны өткізіп жіберіп, келесіге өтеді. Цикл тоқтамайды, тек сол қадам өтіп кетеді.

for i in range(6): if i == 3: continue # 3-ті өткізіп жібер print(i, end=' ') # Нәтиже: 0 1 2 4 5
Break vs Continue:
• break → циклді толықтай тоқтатады
• continue → тек сол итерацияны өткізеді, цикл жалғасады

🔀 For/While + Else

Python-да цикл табиғи аяқталса (break болмаса) — else блогы орындалады. Бұл Python-ның бірегей ерекшелігі!

for i in range(5): print(i) else: print('Цикл сәтті аяқталды!') # else ОРЫНДАЛАДЫ — break жоқ --- for i in range(5): if i == 3: break else: print('Бұл шықпайды!') # else ОРЫНДАЛМАЙДЫ — break болды
📌 Ереже: break → else жоқ. Табиғи аяқталу → else орындалады.

📦 range() функциясы

range() — сандар тізбегін жасайтын кірістірілген функция. For циклімен жиі қолданылады.

# range(stop) range(5) # → 0, 1, 2, 3, 4 # range(start, stop) range(1, 6) # → 1, 2, 3, 4, 5 # range(start, stop, step) range(0, 10, 2) # → 0, 2, 4, 6, 8 range(10, 0, -1) # → 10, 9, 8, ..., 1
⚠️ Маңызды: stop мәні ешқашан қосылмайды!
range(1, 5) = 1, 2, 3, 4 (5 жоқ)

❓ Жиі қойылатын сұрақтар

For мен While айырмашылығы не? ▼
For — неше рет орындалатыны белгілі болғанда. While — шарт орындалып тұрса қайталайды, санды алдын-ала білмеуің мүмкін.
Break пен Continue айырмашылығы? ▼
Break — циклді толықтай тоқтатады. Continue — тек ағымдағы итерацияны өткізіп жіберіп, цикл жалғаса береді.
range(1, 5) неше сан береді? ▼
4 сан: 1, 2, 3, 4. Stop мәні (5) ешқашан қосылмайды — бұл Python-ның маңызды ережесі.
Else циклде қашан орындалады? ▼
Цикл табиғи аяқталса (break болмаса) — else орындалады. Break болса — else орындалмайды.
